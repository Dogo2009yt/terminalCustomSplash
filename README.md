# Terminal Custom Splash

A lightweight Linux terminal customization tool that displays a random splash screen every time you open a new terminal session. 

---

## 🚀 Features

* 🎲 Random splash text generator
* 🎨 Stylish ASCII text
* 🌈 Colored output
* 🐮 Fun terminal mascot using
* 🌦 Optional weather integration
* 🐧 Tux-themed ASCII art support
* ⚡ Auto-start on terminal launch

---

## 📦 Requirements

The installer script automatically installs dependencies, but you can also install them manually:

### Core dependencies

* Python 3
* figlet
* cowsay
* curl
* lolcat (via package manager or Ruby gem)

---

## 🛠 Installation

### 1. Clone or copy the project

```bash
cd ~
git clone https://github.com/Dogo2009yt/terminalCustomSplash.git
cd terminalCustomSplash
```

or manually place it in:

```bash
~/terminalCustomSplash
```

---

### 2. Run the installer

```bash
chmod +x install.sh
./install.sh
```

The installer will:

* Install all required packages
* Configure auto-start in `.bashrc`
